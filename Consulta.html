Acordeon con Bootstrap

Bootstrap fue desarrollado por Twitter en 2011 para unificar el diseño en todas sus plataformas. Desde entonces, se ha convertido en el framework front-end más popular del mundo.

Bootstrap permite desarrollar sitios web responsivos rápidamente gracias a su sistema de grillas, componentes reutilizables y compatibilidad con los principales navegadores.

Un ejemplo común de uso de Bootstrap es la creación de formularios atractivos y funcionales sin necesidad de escribir mucho CSS, gracias a sus clases predefinidas.
